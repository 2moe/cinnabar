## 前言

**問：為何取名為 Cinnabar（硃砂）？**

**答：**

1.  硃砂是中國古代道教煉丹師在煉丹時使用的一種原料，就像這個專案在 CI 工作流中充當一種“原料”。
2.  硃砂有毒。這個專案是為了 **猛、糙、快** (a.k.a. *Dirty and Quick*) 的目的而開發的，可能會產生意料之外的副作用（它並非完全無害）。
3.  硃砂是一種硫化汞 (HgS) 的礦物，呈深紅色，而 Ruby 也是一種深紅色的寶石。給一個 Ruby 專案取名為 “Cinnabar（硃砂）” 非常貼切。
